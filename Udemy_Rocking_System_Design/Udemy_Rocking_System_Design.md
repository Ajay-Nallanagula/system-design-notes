AWS Services to know:

- EC2 - Elastic Compute Cloud (virtual servers)
- S3 - Simple Storage Service (object storage)
- RDS - Relational Database Service (managed SQL databases)
- Elastic Load Balancing (ELB) - Distributes incoming traffic across multiple targets
- CloudFront - Content Delivery Network (CDN)
- Lambda - Serverless compute service
- API Gateway - Managed service for creating and publishing APIs
- Container Services - Amazon ECS (Elastic Container Service),
- EKS (Elastic Kubernetes Service)
